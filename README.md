# Exploratory Data Analysis - Online Learning Activity

## Overview
This project analyzes learner engagement data from an online learning platform to identify patterns in learner behavior and course completion.

## How to Run
Open the Jupyter notebook inside the `code` folder and run all cells from top to bottom.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Notes
Data cleaning was performed to remove unrealistic values and handle missing data. A new feature called days_active was created to measure learner engagement duration.
